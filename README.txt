How to install BlueVikings Software.

You must have three things to run BlueVikings application:

1.- Choose the archive zip file named BlueVikings.zip. For instancethe current version that you are installing supports Windows 64-bit, x86 and MacOs.

2.- Open the location of the \BlueVikings\ExamProject2017\dist  and then then".jar" file (Executable Jar File) and pin to the taskbars or your desktop.
	
3.- Run the application  "ExamProject2017"
